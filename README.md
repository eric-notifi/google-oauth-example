# GOOGLE OAUTH EXAMPLE

## How to get started

1. In the terminal, run `mv .env.local.example .env.local`

2. Fill in Google oauth credentials accordingly in `.env.local`

3. `npm install`

4. `npm run dev`

5. Access `http://localhost:3000`
